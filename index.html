Bienvenu chez CARSIM

Notre site web vous permet de trouver le modèle de voiture parfaitement adapté à vos besoins et à votre budget. En quelques étapes simples, vous pouvez entrer le prix que vous souhaitez dépenser, ainsi que les caractéristiques spécifiques que vous recherchez dans un véhicule. Notre algorithme intelligent parcourt une vaste base de données de voitures pour vous présenter les meilleures options correspondant à vos critères. Que vous recherchiez une voiture économique pour la ville, un véhicule familial spacieux, notre site vous aide à trouver la voiture parfaite pour vous, en fonction de vos préférences et de votre budget.

Conception et implémentation d'un outil d'estimation
des prix des véhicules.

  • La collecte des données : principalement à travers le développement d'un outil de web scraping pour récupérer des données sur des plateformes ayant des données sur les prix des véhicules.
  • Le prétraitement de données : l'état des données collectées est généralement assez médiocre et nécessite un prétraitement avant qu'elles soient utilisées pour entraîner des modèles de machine learning.
  • L'entrainement des modèles : en utilisant des bibliothèques avec des modèles de machine learning implémentés, on cherche à choisir le meilleur modèle pour l'estimation.
  • Le développement d'une interface graphique : une interface web avec principalement un formulaire pour saisir les données d'un véhicule et fournir une estimation de son prix.
